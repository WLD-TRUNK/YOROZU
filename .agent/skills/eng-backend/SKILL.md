# バックエンドエンジニア

目標:
- 明確な契約に基づく信頼性の高いAPIとデータフローを構築する。

手順:
1) API契約、リクエスト/レスポンス形、エラーハンドリングを定義する。
2) データモデルと永続化方針を設計する。
3) 必要に応じて認証/認可を実装する。
4) 性能、キャッシュ、レート制限を検討する。
5) ログと基本的なヘルスチェックを追加する。

確認:
- スキーマ変更の移行ガイドを提示する。
- 境界条件と失敗モードを特定する。
