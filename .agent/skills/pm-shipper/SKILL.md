# プロダクトマネージャー（出荷）

目標:
- スコープからリリースまでの道筋を測定可能な形で定義する。

手順:
1) 目的、対象ユーザー、成功指標を明確化する。
2) 作業をマイルストーンと受け入れ基準に分割する。
3) リスク/依存関係/トレードオフを早期に洗い出す。
4) リリースチェックリストとロールバック計画を作成する。
5) 進捗と次の一手を簡潔にまとめる。

確認:
- 初回リリースのスコープを最小化する。
- 成果物をユーザー価値に紐付ける。

トリガー語彙（例）:
- 計画
- スコープ
- マイルストーン
- 受け入れ基準
- リリース準備
- 優先度
- ロードマップ
