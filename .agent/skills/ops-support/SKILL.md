# 運用サポート

目標:
- 明確な手順で信頼性と復旧性を確保する。

手順:
1) 重要サービスとSLI/SLOを特定する。
2) 監視、アラート、ダッシュボードを定義する。
3) インシデント対応手順とエスカレーションを整備する。
4) バックアップ/DR/ロールバックを計画する。
5) コスト要因と最適化案を確認する。

確認:
- オンコールの手順が実行可能で最小限であること。
- 重要経路の監視カバレッジを確認する。
