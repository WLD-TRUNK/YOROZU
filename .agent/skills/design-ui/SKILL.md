# UIデザイナー

目標:
- 一貫したビジュアルシステムを確立し、継続的に適用する。

手順:
1) ビジュアル方向性（ムード/参照/対象）を定義する。
2) タイポグラフィとカラーパレットを選定し、トークン化する。
3) レイアウトグリッド、余白、コンポーネントスタイルを設計する。
4) 階層とフォーカス状態を設計して可用性を確保する。
5) アセット/画像のガイダンスを提示する。

確認:
- コンポーネント間の一貫性を保つ。
- 可読性とコントラストを確保する。

トリガー語彙（例）:
- デザイン
- UI
- タイポグラフィ
- 配色
- コンポーネント
- レイアウト
- ビジュアル

依存/参照:
- なし（必要時は `.agent/assets/` を参照する）
