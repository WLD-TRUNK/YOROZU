# 🧠 プロンプト管理リポジトリの作成

このワークフローは、AIプロンプトを管理するための専用リポジトリを作成します。

## 手順1: セットアップ // 高速
- **名前決定**: リポジトリ名を決定します（デフォルト: `prompt-management`）。
- **フォルダ作成**: `YOROZU_PROJECT` 配下にフォルダを作成します。

## 手順2: 初期化
- `git init` を実行します。
- `.gitignore` を作成します。

## 手順3: 構成作成
- `prompts/`: プロンプトファイルを格納するディレクトリを作成します。
- `README.md`: プロンプトの整理方法を説明したドキュメントを作成します。
- `templates/`: プロンプトテンプレート用のディレクトリを作成します。

## 手順4: コミットとプッシュ
- 初期ファイルをすべてコミットします。
- `gh repo create` でGitHubリポジトリを作成します。
- 初回コミットをプッシュして完了です。
