# 🏗️ シンプルウェブアプリ構築ワークフロー

このワークフローは、フレームワークを使わずに、標準的なウェブ技術（HTML5, CSS3, ES6+ JavaScript）のみを用いて、美しく機能的なウェブアプリケーションを迅速に構築します。

## 手順1: 🎨 デザインと構造の設計
1. **要件分析**: ユーザーが求めているアプリの機能と「雰囲気」を理解します。
2. **デザイン方針**:
   - **現代的な美観**: 安っぽいデフォルトのスタイルは禁止です。
   - **タイポグラフィ**: Google Fonts（Inter, Poppins, Noto Sans JPなど）を使用します。
   - **配色**: 洗練されたカラーパレット（グラデーション、ダークモード対応など）をCSS変数で定義します。
   - **レイアウト**: Flexbox または CSS Grid を使用したレスポンシブデザインにします。

## 手順2: 💻 コアファイルの実装
以下の3つのファイルをルートディレクトリ（または適切なサブディレクトリ）に作成します。

1. **`INDEX.html`**:
   - セマンティックなHTML構造（header, main, footer）。
   - 必要なメタタグ（viewport, description）。
   - Google Fonts と `STYLE.css`, `SCRIPT.js` の読み込み。

2. **`STYLE.css`**:
   - CSSリセット。
   - ルート変数（`--primary-color`, etc.）の定義。
   - ホバーエフェクトやトランジションを含めたインタラクティブなスタイル。
   - UIパーツ（ボタン、カード、入力フォーム）のスタイリング。

3. **`SCRIPT.js`**:
   - アプリケーションのロジック。
   - DOM操作による動的なUI更新。
   - イベントリスナーの適切な設定。

## 手順3: 👀 ブラウザでの動作検証
1. 作成した `INDEX.html` をブラウザサブエージェントで開きます。
2. ページの表示崩れがないか、インタラクションが正しく動作するかを目視で確認します。
3. 問題があれば修正し、再度確認します。

## 手順4: 📝 完了確認
- 実装したアプリの概要をユーザーに報告します。
